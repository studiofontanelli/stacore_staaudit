ErrorDetail - Up

rappresenta il dettaglio errore restituito in caso di errori di validazione sui campi di input
field (optional)
String il nome del campo per cui è falita la validazione
error (optional)
String l'errore di validazione effettivo
description (optional)
String la descrizione dell'errore

ErrorDto - Up

Rappresenta l'errore restituito da servizio
status (optional)
Integer status http restituito
code (optional)
String code http restituito
description (optional)
String la descrizione dell'errore
errorDetails (optional)
array[ErrorDetail] i dettagli errore

GenericResponse - Up

Risposta di un servizio generico
message (optional)
String Contiene il messaggio eventuale del servizio chiamato

Operazione - Up

rappresenta l'operazione effettuata
codice
String il codice dell'operazione effettuata
entita (optional)
String l'entità oggetto dell'operazione
chiave (optional)
String la chiave (primaria/logica) dell'entità per la quale si è effettuata l'operazione. La chiave è obbligatoria se l'entità è presente
data
Date la data operazione n formato dd/MM/yyyy HH:mm:ss format: date-time
